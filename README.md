# Simple Expense Tracker (Basit Gider Takip Sistemi)
Bu uygulama, Python kullanarak harcamalarınızı kategorize etmenize ve toplam tutarı görmenize olanak sağlar.
## Özellikler
- Harcamaları miktar ve kategori olarak ekleme.
- Verileri `harcamalar.txt` dosyasında saklama.
- Anlık toplam harcama hesaplama.
## Nasıl Kullanılır?
Projeyi bilgisayarınıza indirin ve terminale `python gider_takip.py` yazın.
